Usage
=====

To use resen, simpy enter `resen` at the command line::

    $ resen

This will open the resen tool::

        ___ ___ ___ ___ _  _ 
       | _ \ __/ __| __| \| |
       |   / _|\__ \ _|| .` |
       |_|_\___|___/___|_|\_|
    
    Resen 19.1.0 -- Reproducible Software Environment
    
    [resen] >>> 

Type `help` to see available comands::

    [resen] >>> help
